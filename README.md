# Unity RollABall

A beginner Unity project based on Unity's Roll-a-Ball tutorial.

## Features

* Player movement with keyboard and controller support
* Collectible pickups
* Score tracking
* Win and lose conditions
* Enemy AI using NavMesh pathfinding
* Obstacles that affect enemy movement

## Controls

* WASD to move
* Left stick on controller to move

## Unity Version

Created using Unity 6

## Accessing the Project

This repository contains the Unity project source files and is not a standalone game build.

To run the project:

* Download or clone the repository
* Open the project in Unity Hub
* Use Unity 6 or a compatible version
* Open the main scene and press Play in the Unity Editor
